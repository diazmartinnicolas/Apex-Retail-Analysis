# 📊 Análisis de Ventas - Apex Retail

## 📝 Descripción del Proyecto
Este proyecto simula el trabajo de un **Data Analyst Jr.** para la empresa ficticia "Apex Retail". El objetivo fue transformar un dataset bruto de 5,000 registros en un reporte ejecutivo con insights de negocio.

## 🛠️ Herramientas utilizadas
* **Python**: Lenguaje principal.
* **Pandas**: Limpieza y manipulación de datos.
* **Matplotlib**: Visualización de datos.
* **Jupyter Notebook**: Entorno de desarrollo.

## 🚀 Logros Clave
- **Limpieza de Datos**: Eliminación de duplicados y tratamiento de valores nulos e inconsistencias en precios.
- **Ingeniería de Datos**: Extracción de componentes temporales (días de la semana y horas) para análisis de estacionalidad.
- **Insights de Negocio**: Identificación del jueves como el día de mayores ventas y la categoría "Garden" como la de mayor Ticket Promedio (AOV).
- **Automatización**: Generación automática de reportes multidimensionales en Excel.

## 📂 Estructura del Repo
* `apex_retail.ipynb`: Código completo documentado.
* `Reporte_Ventas_Apex_Retail.xlsx`: Entregable final para gerencia.
